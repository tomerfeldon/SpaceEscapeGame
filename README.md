# üöÄ Space Escape Game

An exciting vertical arcade game for Android where you pilot a spaceship through a treacherous asteroid field. Built entirely using standard Android Views to demonstrate mastery of layouts and dynamic UI manipulation.

## ‚ú® Technologies

* **Kotlin** - Core logic and game mechanics.
* **Android SDK** - Native Android development.
* **XML Layouts** - UI design using `RelativeLayout` and `ConstraintLayout`.
* **Handler & Runnable** - Custom game loop implementation.
* **SharedPreferences** - Local data persistence for High Scores.
* **MediaPlayer** - Audio management for background music and sound effects.

## üöÄ Features

* **Lane-Based Gameplay** - Smooth movement between 3 vertical lanes.
* **Dynamic Difficulty** - The game speed increases as you score higher points.
* **High Score System** - Your best runs are saved locally so you can challenge yourself.
* **Power-Ups & Hazards** - Dodge asteroids (Stones) and collect Coins for bonus points.
* **Immersive Feedback** - Visual explosions, vibration (haptics), and sound effects on collisions.
* **Life System** - 3-heart health system with visual UI updates.


## üö¶ Running the Project

1.  **Clone the repository**
2.  **Open in Android Studio**: Select "Open an Existing Project" and choose the folder.
3.  **Sync Gradle**: Wait for the dependencies to download.
4.  **Run**: Connect an Android device or start an Emulator (AVD).
5.  **Play**: Click the "Run" button.
